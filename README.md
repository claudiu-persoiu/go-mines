## GO-MINES

This game is a replica of https://github.com/claudiu-persoiu/minesweeper, build using Go and compiled to WebAssembly.

The purpose of this project is to demonstrate how to build a simple web game using Go and WebAssembly.

The original game was build using only vanilla JavaScript, HTML and CSS.

### How to run

To compile the code to WebAssembly, run:

```bash
make build
```

To run the server, run:

```bashbash
make run
```

Then open your browser and navigate to `http://localhost:3000`.

### Just what to play now?

You can play the game directly on the following link: https://claudiu-persoiu.github.io/go-mines/
